# NestJS Clean Architecture

## Description

Example of a clean architecture task application with NestJS

## Installation

```bash
$ npm install
```

## Creating .env

```bash
DATABASE_TYPE= database-type
DATABASE_HOST= database-host
DATABASE_PORT= 1234
DATABASE_USERNAME= database-username
DATABASE_PASSWORD= database-password
DATABASE_NAME= database-name
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
